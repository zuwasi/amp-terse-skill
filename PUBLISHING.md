## Publish Commands

Replace `<github-user>` with your GitHub username or org.

### SSH Remote

```bash
git remote add origin git@github.com:<github-user>/amp-terse-skill.git
git branch -M main
git push -u origin main
```

### HTTPS Remote

```bash
git remote add origin https://github.com/<github-user>/amp-terse-skill.git
git branch -M main
git push -u origin main
```

## If Remote Already Exists

```bash
git remote set-url origin git@github.com:<github-user>/amp-terse-skill.git
git push -u origin main
```
